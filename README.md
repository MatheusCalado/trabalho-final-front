# Calado Lubrificantes

## Integrantes da dupla

- Matheus Calado Chaves

## Descrição do caso escolhido

O projeto foi desenvolvido para a Calado Lubrificantes, empresa especializada na comercialização de óleos lubrificantes da marca Panther. O proprietário da empresa é tio do aluno Matheus Calado Chaves. Foi identificado que a empresa não possuía participação no meio digital, não contava com um site próprio e praticamente não utilizava redes sociais para divulgação dos seus produtos e serviços.

Diante dessa necessidade, foi proposta a criação de um site institucional simples, moderno e responsivo, voltado para a divulgação da empresa e para facilitar o contato com os clientes.

## Necessidade identificada

A principal necessidade observada foi a ausência de presença digital da empresa. Os clientes tinham dificuldade em conhecer os produtos comercializados e em encontrar informações básicas sobre a Calado Lubrificantes.

Com a criação do site, buscou-se proporcionar maior visibilidade à empresa, facilitar a divulgação dos produtos da marca Panther e permitir que os clientes entrassem em contato para solicitar orçamentos.

## Público-alvo

O site foi desenvolvido para atender:

- Proprietários de automóveis;
- Proprietários de motocicletas;
- Proprietários de veículos utilitários;
- Oficinas mecânicas;
- Empresas que utilizam lubrificantes automotivos;
- Pessoas interessadas em produtos da marca Panther.

## Objetivo do site

O principal objetivo do site é divulgar a empresa Calado Lubrificantes e seus produtos, além de facilitar o contato com os clientes através do WhatsApp para solicitação de orçamentos.

## Processo de desenvolvimento

Inicialmente foi realizado um levantamento das necessidades da empresa e das informações que deveriam ser apresentadas aos clientes.

Em seguida, foi desenvolvida a estrutura do site utilizando HTML semântico, organizando o conteúdo em seções específicas. Posteriormente, foi realizada a estilização em CSS para proporcionar uma identidade visual moderna, utilizando as cores preta e amarela, características da marca Panther.

Foram adicionadas imagens dos produtos e um banner personalizado para tornar a página mais atrativa. Também foram implementados recursos em JavaScript e Bootstrap para melhorar a experiência do usuário, a responsividade e a navegação do site.

## Principais decisões do projeto

As principais decisões tomadas durante o desenvolvimento foram:

- Utilização de HTML semântico com os elementos header, nav, main, section, article e footer;
- Redução do uso excessivo de divs, utilizando-as apenas quando necessário para componentes do Bootstrap;
- Utilização do Bootstrap para tornar o site responsivo;
- Aplicação de uma identidade visual baseada nas cores da marca Panther;
- Inserção de imagens dos produtos para facilitar a identificação pelo usuário;
- Inclusão de atalhos de teclado e melhorias de acessibilidade;
- Implementação de navegação suave entre as seções da página.

## Funcionalidade em JavaScript

Foi desenvolvido um arquivo JavaScript responsável por adicionar comportamentos ao site.

Entre as funcionalidades implementadas estão:

- Rolagem suave entre as seções da página;
- Navegação por teclado utilizando as teclas direcionais;
- Controle do foco para melhorar a acessibilidade;
- Exibição de mensagens na página;
- Carregamento do Bootstrap por meio do JavaScript.

Essas funcionalidades foram escolhidas para proporcionar uma experiência de navegação mais agradável e acessível ao usuário.

## Uso de Bootstrap

O Bootstrap foi utilizado para facilitar a construção do layout responsivo.

Foram utilizados:

- Navbar responsiva;
- Sistema de grid;
- Cards dos produtos;
- Botões estilizados;
- Classes utilitárias para espaçamento e alinhamento;
- Componentes adaptáveis para diferentes tamanhos de tela.

A utilização do Bootstrap permitiu que o site se adaptasse adequadamente a computadores, tablets e smartphones.
